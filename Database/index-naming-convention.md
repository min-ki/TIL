# DB Naming Convention

## Index Naming Convention
- IX_ : non clustered non unique indexes
- UX_ : for unique indexes

---

회사에서는 네이밍 컨벤션을 보면 아래처럼 쓰고 있는데 뭐가 좋을지 모르겠다..
- tablename_컬럼1_컬럼2_IDX
- tablename_컬럼1_컬럼2_index
- tablename_컬럼1_컬럼2_idx
