# LSM Tree

- log-structured merge-tree (LSM Tree)
- LSM Tree는 다른 검색트리와 마찬가지로 key-value를 유지한다.
- LSM Tree는 key-value를 메모리에 유지하고, 메모리가 꽉 차면 디스크에 쓴다.
